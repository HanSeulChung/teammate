# TeamMate : 팀원들과의 원할한 협업을 위한 서비스
TeamMate는 협업 해야하는 사람들과 팀을 만들어 문서 공유와 캘린더로 일정 공유를 할 수 있어 원활한 협업을 가능하게 합니다.
